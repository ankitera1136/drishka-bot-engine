# Drishka Library Telegram Bot

Drishka Library Telegram Bot is a self-study project built to manage and automate basic library services using Telegram. The bot is designed to make common library tasks easier for users while helping the admin handle routine operations efficiently.

## Features

### User Features
- View library information and working hours
- Search for available books
- Check borrowed books
- Renew books or library membership
- Receive automatic fee and renewal reminders
- Get announcements and offers
- Access payment links
- Submit feedback
- Contact the library admin

### Admin Features
- Broadcast messages to users
- Schedule reminders
- View registered users
- Check basic usage analytics
- Ban users when required (optional)

## Purpose

This project is created mainly for learning and practice. It focuses on understanding Telegram Bot APIs, backend logic, automation, and clean project structure. It can be extended further into a full production-ready system.
